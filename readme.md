## yohe
Sublime Text 3 Plugin。

##### 为sublime Text右侧菜单添加以下功能:


* `Open In Browser`
   自动打开当前项目在apache中对应virtual host。
* `Open In Git`
   在当前目录的打开git。
* `Open In Folder`
   在资源管理器中打开文件目录。

##### 安装
 1.[download](https://github.com/magicnote/yohe/archive/master.zip)文件，打开 Sublime Text 找到 `Preferences` 菜单下的`Browse Packages...`，    
  将下载文件解压后放入打开目录。  
 2.重启 Sublime Text。
